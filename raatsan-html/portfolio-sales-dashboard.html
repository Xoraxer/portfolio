Power BI Project

Sales Distribution Dashboard

In this sales dashboard, we analyze and measure important KPIs in an easy to understand format.

Data Preparation

This sales distribution dashboard analyzes data from an example dataset from Kaggle, so the preparation was minimal.

KPIs

It shows important KPI measures such as monthly sales, top customers, total sales, sales by product, and sales by region.

Design

We are using Microsoft Power BI's various visualization capabilities, such as cards, line charts, pie charts, and lists to create an easy to understand visualization.

Development

We imported the data to Power BI, created a mockup visualziation in a whiteboard, and started to build the KPIs.

Final Delivery

The dashboard can be used to help management evaluate whether or not monthly/yearly sales goals will be met, compare sales results among different time frames and identify if performance is increasing/decreasing.
